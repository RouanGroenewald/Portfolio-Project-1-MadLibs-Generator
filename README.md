# Portfolio-Project-1-MadLibs-Generator
The project is an interactive script that prompts users for various parts of speech (nouns, verbs, adjectives) and injects them into a pre-defined story template. This project focuses on mastering the input() and print() functions, string concatenation, and f-string formatting.
By pushing this to GitHub, it will establish the very first commitment and begining of beginer version-tracked development.
